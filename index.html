// プライバシーポリシー

最終更新: 2025年6月

1. 収集する情報

本サービスは以下の情報を収集します。

  • メールアドレス（アカウント登録時のみ）
  • 学習データ（問題・ミッションのクリア状況、ログイン日時、学習時間）— 主にお使いのデバイスのローカルストレージに保存。ログイン時はSupabaseサーバーにも同期されます。

2. 利用目的

  • ユーザー認証・進捗の保存と同期
  • サービスの品質改善

3. 第三者提供

法令に基づく場合を除き、収集した個人情報を第三者に提供することはありません。

4. 利用する外部サービス

  • Supabase — 認証・データ保存（supabase.com）
  • Wandbox — コード実行（wandbox.org）。入力したコードは実行のためWandboxに送信されます。
  • Groq — AI機能（groq.com）。AIヒント使用時にコードと質問内容が送信されます。
  • Vercel — ホスティング（vercel.com）

5. お問い合わせ

プライバシーに関するご質問は GitHub Issues よりお問い合わせください。
